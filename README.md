# Speedoku Royale | Front

Web site for hosting the game

## Prerequisites

- VS Code
- Prettier extension for VS Code
- NPM
- Yarn

After you have checked that you have these, run `yarn install` while being in
the project root.

## Commands
 Command      | Description
--------------|------------------------------
 `yarn dev`   | Start the development server
 `yarn build` | Builds the project and puts the result into **dist** folder
 `CTRL + C`   | Closes the development server
